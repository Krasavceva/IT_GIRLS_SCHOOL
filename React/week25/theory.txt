2. Перечислите плюсы и минусы SPA(Single Page Applications) приложений?
    Плюсы:
 - Скорость работы (т.к. это всего 1 страница и не нужно тратить доп.время на повторную загрузку данных).
 - Возможность задействовать большие объемы данных.(Размер приложения и используемых им данных не ограничен памятью устройства).
 - SPA работают везде, где есть поддержка JavaScript. 
 - Возможность быстрой разработки, т.к. доступны фреймворки, которые упрощают создание архитектуры проекта и предоставляют немало готовых элементов для работы.
    Минусы:
 - Необходимость интернет-соединения. Без доступа к сети использовать такой софт невозможно. 
 - Трудности с SEO. Особенности SPA усложняют или делают невозможным процесс индексации поисковыми системами всех модулей приложения. Это может вызвать трудности с оптимизацией.
 - Не работает у пользователей с отключенной поддержкой JS. Многие отключают отображение JS-элементов у себя в браузерах, а Single Page Application использует их в работе, поэтому может не работать.

4. Является ли React фреймворком? А какие есть еще веб-фронтенд фреймворки?
React не является фреймворком, это библиотека. Он используется для визуализации и в связке с другими библиотеками.
Но в разговорной речи React называют фреймворком.
Самые популярные веб-фреймворки: React, Angular и Vue.
